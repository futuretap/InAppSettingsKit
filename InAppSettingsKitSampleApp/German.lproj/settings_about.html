Diese View zeigt Informationen zum Programm, Lizenzbedingungen oder andere Dinge, die in einer WebView angezeigt werden sollen.

mailto links werden automatisch abgefangen und an eine inline Mail Compose View weitergereicht: mailto:foo@bar.com?subject=FooBar&body=foo.

Links mit dem selben Host werden inline dargestellt; wenn der host unterschiedlich ist, werden sie ans System (sprich: Safari für http-Links) weitergereicht.

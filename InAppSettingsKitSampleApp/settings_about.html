About

This view allows you to display about information, license terms, or anything else you want to show in a web view.

It is only available in in-app settings.

The keys and values are:

  • Type: IASKWebViewSpecifier
  • Title: The title displayed in the settings view.
  • File: Either the name of the HTML file in the application's bundle or the URL to an HTML file on a server.
  • BaseURL: If the HTML file is on a remote server you should specify the base URL for any links in the HTML file.
